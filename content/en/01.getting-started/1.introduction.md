## What is XMPlus?

XMPlus is an xray-core proxy user subscription panel and server to manage traffic and ip connections. Writen with **Laravel** and **Nuxt** framework.

## Key Features

This panel (frontend UX and backend API) includes a range of features :

- **UX Powered by** **[Nuxt 4](https://nuxt.com)**: Utilizes the latest Nuxt framework for optimal performance.
- **Built with** **[Vuetifyjs](https://vuetifyjs.com)**: Integrates a comprehensive suite of UI components.
- **API Powered by** **[Larave 12](https://laravel.com/docs/12.x)**: Provides a complete ecosystem for web artisans.
- **[Server](https://github.com/XMPlusDev/XMPlusServer)** **built with** [Xray-core](https://github.com/XTLS/Xray-core): An anti-censorship tool that penetrate everything.
- **IP Limit**: The ip limit feature is implemented using redis and golang sync Map.
- **Speed Limit**: Independent connection port speed limit for each subscription.
- **Payment Method**: The panel integrate payments such as **[stripe](https://www.stripe.com)** and **[alipay](https://www.alipay.com)** gateways.
