## Requirements

::warning
Installed on top of a fresh operating system to ensure proper functionality.
::

|          | Minimum                                    | Recommended                        |
| -------- | ------------------------------------------ | ---------------------------------- |
| **CPU**  | 2 core, 64-bit                             | 4 cores                            |
| **RAM**  | 2 GB                                       | 4 GB or more                       |
| **Disk** | 20 GB HDD                                  | 40 GB SSD                          |
| **OS**   | Debian 11 or 12 or Ubuntu 22.04, 24.04 LTS | Latest Debian or Latest Ubuntu LTS |

::steps
### Step 1: System Update

Update system repository and install packages:

```bash
apt-get update -y  && apt-get install wget bash zip curl unzip git ca-certificates -y
```

### Step 2: Install Docker

```bash
curl -fsSL https://get.docker.com -o get-docker.sh && sh get-docker.sh
```

### Step 3: Install Docker Compose

```bash
sudo curl -L "https://github.com/docker/compose/releases/download/v5.1.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

sudo chmod +x /usr/local/bin/docker-compose

sudo ln -s /usr/local/bin/docker-compose /usr/bin/docker-compose 
```

### Step 4: Download files

```bash
wget -N --no-check-certificate -O /home/XMPlus.zip https://github.com/XMPlusDev/XMPlusRelease/releases/download/v2603130/XMPlus-Panel.zip && cd /home && unzip XMPlus.zip


```

```bash

DB_USERNAME=xmplus

DB_ROOT_PASSWORD=76Z1RCLYgg7UOU1QQ63z  # change to any value of your chosen

REDIS_PASSWORD=7QkUQCM9pD7G2uuOeONv  # change to any value of your chosen
REDIS_PORT=6379

REVERB_APP_KEY=BKy5lJIXaGwTW2CRD6Vc    # change to any value of your chosen
REVERB_APP_SECRET=9ymdOudYDr8RQWoOp9yN # change to any value of your chosen

REVERB_HOST=api.tld.com    # change to the host address your want to use and make sure dns resolves to the hosting server
```

### Step 6: Modify API\_URL in ecosystem.config.cjs

```bash
module.exports = {
  apps: [
    {
      name: 'XMPlus',
      script: '.output/server/index.mjs',
      instances: 'max',
      exec_mode: 'cluster',
      watch: false,
      env: {
        API_URL: 'https://api.tld.com',
        PORT: 3005,
        DEBUG: false,
        SESSION_HTTPONLY: true,
        SESSION_SECURE: true,
        SESSION_SAME_SITE: 'lax'
      }
    }
  ]
}

```

### Step 7: Run Docker Composer

  :::warning
  You must allow ports `80, 443, 81, 6379 and 8081` through firewall or iptables
  :::

```bash
cd /home/XMPlus-Panel

# Pull and run services
docker-compose up -d

# migrate initial database tables
docker compose exec api php artisan migrate --seed

# create admin account
docker compose exec api php artisan xmplus:create-admin-account
```

    # Laravel Reverb WebSocket Server
      location /app {
          proxy_pass http://api:8080;
          proxy_http_version 1.1;
          proxy_set_header Upgrade $http_upgrade;
          proxy_set_header Connection "Upgrade";
          proxy_set_header Host $host;

          proxy_cache_bypass $http_upgrade;
          proxy_read_timeout 86400s;
          proxy_send_timeout 86400s;
          proxy_connect_timeout 5s;
        
      # Disable buffering for WebSockets
          proxy_buffering off;
          proxy_request_buffering off;
      }

      # Laravel Reverb health check and broadcasting endpoints
      location ~ ^/(health|apps|events) {
          proxy_pass http://api:8080;
          proxy_http_version 1.1;
          proxy_set_header Host $host;
          proxy_set_header X-Real-IP $remote_addr;
          proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
          proxy_set_header X-Forwarded-Host $host;
          proxy_set_header X-Forwarded-Proto $scheme;
      }

      # Deny access to sensitive files
      location ~ /\.(?!well-known) {
          deny all;
          return 404;
      }

      location ~ /\.env {
          deny all;
          return 404;
      }


      # Main application - proxy to Laravel Octane
      location / {
          try_files $uri $uri/ @octane;

          # Static files caching
          location ~* \.(?:jpg|jpeg|gif|png|ico|svg|webp|css|js|woff|woff2|ttf|eot|otf)$ {
              expires 1y;
              access_log off;
              add_header Cache-Control "public, immutable";
              try_files $uri @octane;
          }
      }

      # Proxy all requests to Laravel Octane
      location @octane {
          proxy_pass http://api:9000;
          proxy_http_version 1.1;
          
          # Headers for Octane
          proxy_set_header Host $host;
          proxy_set_header X-Real-IP $remote_addr;
          proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
          proxy_set_header X-Forwarded-Proto $scheme;
          proxy_set_header X-Forwarded-Host $host;
          proxy_set_header X-Forwarded-Port $server_port;
          
          # Connection settings for Octane
          proxy_set_header Connection "";
          proxy_buffering off;
          proxy_request_buffering off;
          
          # Timeouts
          proxy_connect_timeout 60s;
          proxy_send_timeout 60s;
          proxy_read_timeout 60s;
      }

      # Block access to storage and bootstrap/cache
      location ~* ^/(storage|bootstrap/cache) {
          deny all;
          return 404;
      }
      
        location /index.php {
          try_files /not_exists @octane;
      }

      error_page 404 /index.php;

      location ~ /\.ht {

### Accesss website in your browser <https://www.tld.com>

To access phpmyadmin <http://your-server-ip:8081>.

Use database login details set inyour .env file.

  :::caution
  Disable port 8081 if not using phpmyadmin at any point in time to prevent unauthorize spoofing access.
  :::

```bash
cd /home/XMPlus-Panel

docker-compose down

docker-compose Pull

docker-compose up -d

```
