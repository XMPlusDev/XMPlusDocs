# Application Configuration

## Endpoint Overview

| Property          | Value              |
| ----------------- | ------------------ |
| **Method**        | `POST`             |
| **URL**           | `/api/config`      |
| **Content-Type**  | `application/json` |
| **Accept**        | `application/json` |
| **Auth Required** | No (Public)        |
| **Request Body**  | *(none)*           |

> This endpoint takes no request body or headers. It returns the application's public configuration settings.

---

## Response

### `200 OK` — Success

#### General Settings

| Field             | Type     | Example Value             | Description                                  |
| ----------------- | -------- | ------------------------- | -------------------------------------------- |
| `app_name`        | `string` | `"XMPlus"`                | The application display name.                |
| `app_description` | `string` | `"Network Optimizer"`     | Short description of the application.        |
| `app_timezone`    | `string` | `"UTC"`                   | Default timezone used by the application.    |
| `app_locale`      | `string` | `"en"`                    | Default locale/language code.                |
| `app_logo`        | `string` | `"uploads/logos/..."`     | Relative path to the application logo image. |
| `app_favicon`     | `string` | `"uploads/favicons/..."`  | Relative path to the application favicon.    |
| `frontend_url`    | `string` | `"http://localhost:3000"` | The base URL of the frontend application.    |

#### Maintenance & Security

| Field                | Type      | Example Value | Description                                       |
| -------------------- | --------- | ------------- | ------------------------------------------------- |
| `force_https`        | `boolean` | `false`       | Whether the app forces HTTPS connections.         |
| `maintainance_mode`  | `boolean` | `false`       | Whether the app is currently in maintenance mode. |
| `maintainance_note`  | `string`  | `""`          | Message displayed to users during maintenance.    |
| `enable_restriction` | `boolean` | `false`       | Whether access restrictions are enforced.         |

| ----------------------- | --------- | ------------- | ------------------------------------------------------ |

| `login_authentication`  | `boolean` | `false`       | Whether two-factor authentication is enabled at login. |

| Field                 | Type      | Example Value | Description                                                |
| --------------------- | --------- | ------------- | ---------------------------------------------------------- |
| `remote_translations` | `boolean` | `false`       | Whether translations are loaded from a remote source.      |
| `isshowLocales`       | `boolean` | `true`        | Whether the locale/language switcher is visible in the UI. |

| Field     | Type      | Description                                      |
| --------- | --------- | ------------------------------------------------ |
| `locale`  | `string`  | Locale code (e.g. `"en"`, `"fa"`, `"zh"`).       |
| `lang`    | `string`  | Human-readable language name (e.g. `"English"`). |
| `country` | `string`  | Two-letter country code for flag display.        |
| `isRTL`   | `boolean` | Whether the language is right-to-left.           |

#### Live Chat

| Field              | Type      | Example Value | Description                                |
| ------------------ | --------- | ------------- | ------------------------------------------ |
| `show_livechat`    | `boolean` | `true`        | Whether the live chat widget is shown.     |
| `tawk_status`      | `boolean` | `false`       | Whether Tawk.to integration is active.     |
| `tawk_all_page`    | `boolean` | `false`       | Whether Tawk.to chat appears on all pages. |
| `tawk_property_id` | `string`  | `""`          | Tawk.to property ID.                       |
| `tawk_widget_id`   | `string`  | `""`          | Tawk.to widget ID.                         |

#### Miscellaneous

| Field            | Type      | Example Value | Description                                           |
| ---------------- | --------- | ------------- | ----------------------------------------------------- |
| `renewal_type`   | `integer` | `1`           | Subscription renewal type identifier.                 |
| `legacy_sublink` | `boolean` | `true`        | Whether legacy subscription link format is supported. |

## Code Sample — PHP (cURL)

```php
<?php
$curl = curl_init();

curl_setopt_array($curl, [
  CURLOPT_URL            => 'http://127.5.5.5/api/config',
  CURLOPT_RETURNTRANSFER => true,
  CURLOPT_FOLLOWLOCATION => true,
  CURLOPT_HTTP_VERSION   => CURL_HTTP_VERSION_1_1,
  CURLOPT_CUSTOMREQUEST  => 'POST',
]);

$response = curl_exec($curl);
curl_close($curl);

$config = json_decode($response, true);
echo $config['app_name']; // "XMPlus"
```
