---
title: Cloudflare - API Token
description: Cloudflare api credentials for dns zones
---

Create a cloudflare account from [Cloudflare](https://dash.cloudflare.com/login)

Navigate to [User API Tokens](https://dash.cloudflare.com/profile/api-tokens) and create **API Tokens**. Use **Edit zone DNS** Template.

Modify `Zone Resources` and select **All Zones**

Click on **Continue to summary** an **Create Token**. Copy and keep secure your token

::code-preview
![Add new user](/images/cloudflare_token.png)
::

> **Cloudflare API Token**: Recommended to use instead of Cloudflare API Key which does not limit access.

> **Cloudflare API Base URL**: The currect cloudflare api address <https://api.cloudflare.com/client/v4/>
