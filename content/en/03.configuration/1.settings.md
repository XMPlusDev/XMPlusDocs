### Site Information

> **Panel Frontend Url:** This is the host address of your frontend UX with schema. eg, `https://www.tld.com`

> **Panel Backend Url:** This is the laravel api host address. eg, `https://api.tld.com`

> **Authorization Key:** API authorization key.Only required if you have more than 20 users or 50 subscriptions.

> **Remote Translations:** Whether to use remote tranlation from the api. [Read more here](/en/others/translations)

> **IP2Location Token:** IP2Location api token require for updating ip database. [IP2Location API](https://lite.ip2location.com/)

### API Configuration

> **Server API Key:** API Key use to connect XMPlus server(node) to panel backend laravel api.

> **Update Interval (sec):** Time in seconds after which server(node) get and report data to panel(laravel) api. Least must be **60**.
